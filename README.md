# 👟 ShoeShop — Интернет-магазин обуви

Интернет-магазин обуви с полным функционалом: авторизация, корзина, заказы, админ-панель.

## 🚀 Технологии

- **Python 3.8** + **Django 4.2**
- **MySQL** — база данных
- **Bootstrap 5** + **Font Awesome** — дизайн
- **Git** — контроль версий

## 📋 Функционал

### Для всех пользователей:
- Просмотр товаров с фото, ценой и скидками
- Подсветка товаров со скидкой > 45%

### Для авторизованных пользователей:
- Добавление товаров в корзину
- Оформление заказа
- Личный кабинет с редактированием профиля
- Просмотр истории заказов

### Для менеджера:
- Поиск, фильтрация и сортировка товаров
- Просмотр всех заказов

### Для администратора:
- Добавление/редактирование/удаление товаров
- Управление заказами (статусы, удаление)
- Доступ к админ-панели Django

## 🔧 Установка и запуск

```bash
# Клонируйте репозиторий
git clone https://github.com/Kinpil/Shoeshop.git
cd Shoeshop

# Создайте виртуальное окружение
python -m venv venv
venv\Scripts\activate  # Windows

# Установите зависимости
pip install -r requirements.txt

# Примените миграции
python manage.py migrate
python manage.py createsuperuser

# Запустите сервер
python manage.py runserver
